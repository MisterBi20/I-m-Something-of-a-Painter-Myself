## 💡 关键技巧

### 快速提升成绩的方法：

1. **使用数据增强** ⭐⭐⭐
   ```bash
   --diffaugment color translation cutout
   ```

2. **增加训练轮数** ⭐⭐
   ```bash
   --train_epochs 50
   ```

3. **监控训练** ⭐⭐
   ```bash
   --wandb
   ```

4. **调整超参数** ⭐
   - Batch size: 根据显存调整
   - Learning rate: 从2e-4开始

## ⚠️ 常见陷阱

1. **不要跳过baseline**
   - 先确保流程正确
   - 再优化性能

2. **保存每次实验**
   - 每个训练创建新文件夹
   - 保留checkpoint

3. **验证数据格式**
   - 图像数量: 7000-10000
   - 图像格式: JPEG
   - 命名: 1.jpg, 2.jpg, ...

4. **显存管理**
   - 如果OOM，减小batch size
   - 监控GPU使用率
